#2.8 효율성과 가독성 높이기
#금융 자산의 현재 가치는 다음과 같이 계산됩니다.
#PV=FV/(1+r)n
#즉, 1년 뒤에 110만원을 받는 돈의 현재가치는 110만원/(1+0.1)1=100만원 이라 볼 수 있습니다. 이러한 값을 구하기 위해 매번 계산기를 사용하기 보다는 함수를 이용하면, 훨씬 효율적인 작업이 가능합니다. 위의 수식을 함수로 나타내면 다음과 같습니다.
#1.
PV = function(FV, r, n) {
  PV = FV / (1+r)^n
  return(round(PV, 2))
}


#2.
body(PV)
{
  PV = FV/(1 + r)^n
  return(round(PV, 2))
}
